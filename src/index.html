⚠ SMART APP CONTROL WARNING

Smart App Control is active.

Windows Smart App Control blocks unsigned applications and modifications. Since Radium uses custom game patches, Smart App Control may prevent the game from starting or silently block game files.

If the game fails to open, you will need to add exclusions or turn off Smart App Control manually in your Windows Security settings.

⚠ MISSING GAME PATCH FILE

Critical game file is missing: Radeon.Core.BasePatch.dll

The patch DLL was not found in your game directory. This is usually caused by Windows Defender or other antivirus software removing it as a false positive.

Without this DLL, you will not be able to connect to the game servers. Would you like to restore the file now?

Tip: After restoring, we highly recommend clicking "Exclude AV" in settings to prevent Windows Defender from deleting it again.
